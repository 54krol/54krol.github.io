История Интернета

История Интернета началась в 1950-х годах. Впервые принципы по которым строится Интернет были применены в сети ARPANET, использовавшейся американским военным агентством. В 1984 году была создана сеть NSFNET, которая использовала наработки ARPANET и была более доступной. За счет этого в 1992 к этой сети было подключено более 7500 мелких сетей.

В 1984 году появилась FIDOnet - компьютерная сеть использующая модемы и телефонные сети.

Изначально, сеть из которой появился современный интернет, предназначалась для использования правительством. В 1980-х годах исследования Тима Бернерса-Ли позволили создать Всемирную паутину, за счет соединения документов в систему, доступную из любой части сети.

История HTML

HTML был разработан в 1986-1991 годах британским учёным Тимом Бернерсом-Ли. Он создавался как язык который будет пригоден для использования людьми не являющимися специалистами вёрстки. Первым описанием HTML стал документ "Теги HTML" упомянутый в интернете в 1991 году.

Первая версия HTML не имела официальной спецификации. До 1995 года в сети было множество неофициальных стандартов HTML. Основные производители брузеров — Microsoft и NetScape — начали внедрять свои собственные элементы в HTMl разметку. Это создало путаницу из различных конструкций для создания сайтов. Разработчикам приходилось создавать несколько вариантов страниц для разных браузеров. Для того чтобы официальная версия отличалась от них, ей сразу был присвоен второй номер.

Тим Бернерс-Ли

Тим Бернерс-Ли — один из создателей первого браузера "WorldWideWeb" и создатель HTML. Основал и возглавляет Консорциум Всемирной Паутины (W3C), организацию, разрабатывающую стандарты всемирной паутины.

Марк Андриссен

Марк Андриссен — соавтор первого браузера (Mosaic) и основатель компании Netscape Communications. Хотя Mosaic не был первым графическим браузером, но стал первым браузером на который обратили внимание. Также Mosaic стал первым браузером, отображающим изображения внутри текста. Браузер компании Netscape Communications — Netscape Navigator позволил донести преимущества интернета до простых пользователей.

Брайан Белендорф

Брайан Белендорф — ведущий разработчик веб-сервера Apache, один из сооснователей Apache Group (Apache Software Foundation). Основное достоинство веб-сервера Apache — возможность расширения сервера. Это значит, что любой пользователь Apache может добавить расширения (плагины) для улучшения работы сервера. Apache до сих пор является самым популярным HTTP-сервером в Интернете.
